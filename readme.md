# IGNIS

**Intervention Graded Nullifying Incendiary System**  
A field-deployable hazard neutralization apparatus, implemented as a real-life, modular bomb puzzle game.

IGNIS is inspired by *Keep Talking and Nobody Explodes* but uses a **physical bomb puzzle** featuring **swappable riddle modules**.  

Also known as _Kinetic Threat Anomaly Neutralization Equipment_.

## Features
- Physical, real-life modular puzzle bomb
- Swappable riddle modules
- Electronics, firmware, and mechanical domains
- 3D-printed and PCB-based components
- TVA-inspired retro-industrial aesthetic
- Designed for replayability through randomized environment parameters

## Disclaimer

IGNIS contains no explosive or hazardous materials and is purely a puzzle device.
